## Predictive Modeling
Applying supervised machine learning models for predictions and model selection\
Author: Valeria Calderon\
Created: 12/04/2025

<sub> Project done for DS501 in UKY</sub>

Project is about building the most accurate machine learning model that predicts housing prices (regression) and a \
loan default risk (classification) using real world data.

### Models used:
+ Linear Models: Linear Regression
+ Tree-Based Models: Decision Tree, Random Forest
+ Instance-Based Models: K-Nearest Neighbors (KNN)
+ Kernel Methods: Support Vector Machine (SVM)

### Datasets are open source datasets:
+ Ames Housing Dataset
+ Default of Credit Card Clients Dataset
