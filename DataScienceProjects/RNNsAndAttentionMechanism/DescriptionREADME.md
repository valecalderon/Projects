## RNNs and Attention mechanisms
Author: Valeria Calderon
Created: 12/10/2026

<sub> Project done for DS501 in UKY </sub>

This project is for implementing mathematical equations of sequence models. Code includes a working training /
loop and a data generator.

### Vanilla RNN
Implements the recurrence formula to generate text (Shakespeare)

### LSTM (upgraded)
Utilizes PyTorch nn.LSTM to solve vanishing gradients

### Neural Translation with Attention
Implements Bahdanau Attention to allow decode to "look back" at specific parts of the input sentence.
