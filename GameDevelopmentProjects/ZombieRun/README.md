## Zombie Run 🧟
Author: Valeria Calderon\
<sub>All rights reserved</sub>

Zombie Run is a game where you have to escape a zombie by collecting all your limbs and getting to safety, the game has 3 levels and the safety is somewhere in the arena you have to find it. Arrows help find limbs.
Each level the zombie is faster.

Knowledge:

+ Discrete rigid body physics
+ Collision detection
+ Tilemaps
+ Physics in 2D
+ Behavior trees
