## Soduku Cube

Sodoku cube is a simple GUI for a sodoku cube, basically like a Rubik's Cube but
harder, to solve each face you have to solve like in sodoku. This GUI represent the
2D version of it and can randomize each face with user input.

More in depth description of project in: [Description PDF](https://github.com/valecalderon/Projects/blob/main/SoftwareDevelopmentProjects/SodokuCube/SudokuCubeUseDescription.pdf)
